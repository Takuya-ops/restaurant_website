# Food Place - レストラン検索・予約サイト

Figmaデザインに基づいて実装したレストラン検索・予約サイトです。

## ファイル構成

```
Restaurant_website/
├── index.html          # トップページ
├── store-details.html  # ストア詳細ページ
├── styles.css          # スタイルシート
├── script.js           # JavaScript機能
├── Asset/              # アセットファイル
│   ├── Icon/          # アイコンSVGファイル
│   ├── Top/           # トップページ用画像
│   └── Store Details/ # ストア詳細ページ用画像
├── Fonts/             # Interフォント
│   └── Inter/
│       └── static/
└── Design/             # デザインファイル
```

## 機能

### トップページ (index.html)
- ヘッダー（ロゴ、お気に入り、履歴、ログイン/会員登録）
- メインバナー（プロモーション情報）
- 検索・フィルター機能（エリア、キーワード、時間、予約日、人数）
- 近くのレストラン一覧（4つのレストランカード）
- AIチャットボタン（フローティング）

### ストア詳細ページ (store-details.html)
- レストラン情報ヘッダー
- 画像ギャラリー（メイン画像とサムネイル）
- タブナビゲーション（概要、メニュー、座席、写真、アクセス、クチコミ）
- コース一覧
- 座席一覧
- クチコミ一覧
- 店舗情報
- 予約ウィジェット（サイドバー）
  - 電話番号
  - 人数選択
  - カレンダー（予約日選択）
  - 開始時間選択
  - 予約ボタン

## 技術スタック

- HTML5
- CSS3（カスタムフォント、Flexbox、Grid）
- JavaScript（Vanilla JS）

## 使用方法

1. `index.html`をブラウザで開いてトップページを表示
2. レストランカードをクリックして詳細ページに遷移
3. 検索機能や予約ウィジェットを使用

## 主な機能実装

- **タブ切り替え**: ストア詳細ページのタブナビゲーション
- **お気に入り機能**: レストランカードのお気に入りボタン
- **ギャラリー画像切り替え**: サムネイルクリックでメイン画像を変更
- **検索機能**: 検索条件の入力と検索実行
- **カレンダー**: 予約日の選択
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応

## フォント

Interフォントを使用しています。以下のフォントウェイトが利用可能です：
- Regular (400)
- Medium (500)
- SemiBold (600)
- Bold (700)

## ブラウザ対応

モダンブラウザ（Chrome、Firefox、Safari、Edge）に対応しています。
